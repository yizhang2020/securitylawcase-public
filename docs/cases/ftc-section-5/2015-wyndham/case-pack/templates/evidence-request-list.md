# Evidence Request List (48-hour production) — FTC v. Wyndham Worldwide Corp.

Use this checklist when responding to **FTC document requests**, **assessor evidence lists**, or **internal audit** pulls related to the **stipulated order**.

---

## Program and governance
- [ ] Written information security program (current version + change history)
- [ ] Security policies/standards for connectivity, remote access, logging, IR
- [ ] Risk assessments for payment card environments (last 24 months)

## Connectivity and segmentation
- [ ] Network diagrams for property-to-corporate paths (controlled distribution)
- [ ] Firewall / routing configurations for in-scope segments (exports)
- [ ] Connection approval tickets and exception register

## Identity and access
- [ ] MFA policy and enforcement reports for privileged groups
- [ ] PAM configuration summary and session samples (redacted)
- [ ] Access review attestations for administrative accounts

## Monitoring and incidents
- [ ] SIEM coverage documentation and representative alert samples
- [ ] Incident timelines for material events (redacted)
- [ ] Post-incident remediation tracking

## Assessments
- [ ] PCI DSS–oriented assessment reports (per order)
- [ ] Management responses and remediation evidence
