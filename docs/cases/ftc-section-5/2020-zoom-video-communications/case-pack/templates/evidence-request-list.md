# Evidence Request List Template (Zoom FTC Matter)

Use this template to request order-compliance evidence from Security, Product Engineering, Legal, Privacy, Communications, and GRC.

| Evidence area | Requested artifact | Owner | Review cadence |
|---|---|---|---|
| Security claims | Inventory of external and in-product encryption/security claims | Legal / Communications | Quarterly |
| Architecture validation | Cryptography and key-management review records | Security Architecture | Per major release |
| Software updates | Security review and release approval tickets | Product Engineering | Per release |
| Vulnerability management | Open findings, SLA compliance, remediation exceptions | Security Engineering | Monthly |
| Access controls | MFA coverage and privileged-access review | IT / IAM | Monthly |
| Data protection | Cloud-recording encryption, storage, and deletion evidence | Product / Data Platform | Quarterly |
| Incident notification | Covered incident decision logs and tabletop results | Incident Response / Legal | Semiannual |
| Assessment readiness | Independent assessment evidence index | GRC | Biennial / rolling |
