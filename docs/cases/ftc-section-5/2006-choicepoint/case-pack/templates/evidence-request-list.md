# Evidence Request List (FTC v. ChoicePoint)

Use this list to respond to FTC examiners, auditors, or litigation discovery.

## Provide within 48 hours
- Written information security program and program coordinator designation
- Risk assessment and risk register (relevant entries)
- Access control and MFA evidence (privileged/sensitive accounts)
- Repository scanning results and policy (no credentials in code)
- Data retention schedule and deletion/de-identification evidence
- Detection rules, log coverage, and sample investigation outcomes
- Security training records and biennial assessment report (if due)
- Consent order compliance index and milestone tracking
