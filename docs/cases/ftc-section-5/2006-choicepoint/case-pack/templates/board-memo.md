# Board / Executive Memo Template (FTC v. ChoicePoint)

---

## Purpose

Provide a reusable board-level memo structure for reporting information security program status and FTC consent order compliance after the ChoicePoint settlement pattern—aligned with board security brief expectations.

---

## Hallucinated writing examples

Scenario: In November 2022, following the FTC’s consent order (time), the CISO (role) provides a board memo template (type) to a board director (Audit Committee) (audience) for use in future board reporting on program status and consent order compliance.

<div class="writing-example-formal">

<p><strong>MEMORANDUM (TEMPLATE)</strong></p>

<div class="doc-meta">
<p><strong>To: </strong>Board Audit Committee<br>
<strong>From: </strong>Chief Information Security Officer<br>
<strong>Date: </strong>[Date]<br>
<strong>Subject: </strong>Information security program and FTC consent order compliance — status and governance</p>
</div>

<p><strong>Executive Summary (5 Sentences): </strong></p>
<p>- What happened (breach summary: credential reuse, GitHub, exfiltration; discovery via external reports).<br>
- Why it matters (FTC order, consumer trust, individual CEO obligations).<br>
- What we’re changing (program, MFA, credential management, data retention, biennial assessment).<br>
- How we’ll measure effectiveness (program metrics, assessment findings, order milestones).<br>
- What we need from leadership (designation, budget, oversight).</p>

<p><strong>Current Posture (Bullets): </strong><br>
- Information security program (written program, coordinator, risk assessment).<br>
- Access control and credential management (MFA, no credentials in code).<br>
- Data minimization and retention schedule.<br>
- Monitoring and biennial independent assessment.</p>

<p><strong>Metrics (What We’ll Report Quarterly): </strong><br>
- Program coordinator designation and reporting line.<br>
- MFA coverage and repository scanning results.<br>
- Retention schedule compliance (deletion/de-identification).<br>
- Biennial assessment status and finding closure.</p>
