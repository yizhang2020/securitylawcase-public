# Evidence Request List Template (TikTok v. Garland)

Use this template to request evidence for platform-control and foreign-adversary risk review.

| Evidence area | Requested artifact | Owner | Review cadence |
|---|---|---|---|
| Data inventory | Sensitive U.S. user data map and collection purpose | Privacy Engineering | Quarterly |
| Cross-border access | Access paths, transfer logs, exceptions, and approvals | Security / Privacy | Monthly |
| Algorithm control | Recommendation and moderation change-control records | Product Engineering | Per release |
| Source-code custody | Code ownership, dependency, and deployment evidence | Engineering | Per major release |
| Privileged access | Admin access review and MFA evidence | IAM / Security | Monthly |
| Divestiture readiness | Control-separation architecture and dependency map | Legal / Engineering | Program milestone |
| Governance | Board materials, risk acceptance, and escalation logs | GRC / Legal | Quarterly |
