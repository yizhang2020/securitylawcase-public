# Evidence Request List

Use this list to respond to auditors, regulators, or litigation discovery.

## Provide within 48 hours
- Architecture diagram + boundary inventory
- WAF baseline policy + change history
- IAM role inventory + access review evidence
- Cloud audit logs coverage statement + retention config proof
- Detection rules and investigation ticket samples
- Risk register entries + mitigation tracking
- Audit/assessment reports + closure evidence
- Incident response plan + tabletop records
